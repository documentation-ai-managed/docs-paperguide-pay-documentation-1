## Getting Started

1. Sign in at `https://dashboard.paperguide-pay.com`
2. Create a new documentation project 
3. Choose a template or start blank
4. Add your sections and pages
5. Click "Publish" to make docs live

## Tips

- Use Markdown for formatting
- Add team members for collaboration
- Find more options under Project Settings
