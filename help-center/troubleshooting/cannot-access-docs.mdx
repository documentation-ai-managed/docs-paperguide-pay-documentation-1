---
title: Cannot Access Documentation
description: Quick fix for being unable to open documentation.
---

## Symptoms

- Docs page shows a 404 error
- Access denied message

## Possible Causes

- Incorrect URL
- You lack view permissions

## Solution

<Steps>
  <Step title="Check the documentation URL">
    Ensure the link is correct and project is published.
  </Step>
  <Step title="Request access">
    Ask the project owner to grant you view permissions from the team settings.
  </Step>
</Steps>
