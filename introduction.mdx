## Overview

Paperguide-pay empowers you to create, organize, and manage your project's documentation effortlessly. As a SaaS platform, it provides intuitive tools for teams to collaborate on docs, embed interactive elements, and maintain up-to-date knowledge bases. You access everything through a clean dashboard at `https://dashboard.example.com`, with API integrations available at `https://api.example.com`.

Whether you build software, manage internal wikis, or document processes, Paperguide-pay streamlines your workflow. You start by signing up, creating spaces, and inviting collaboratorsâ€”no complex setup required.

## Getting Started

Follow these steps to launch your first project.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://dashboard.example.com/signup` and create your account.
  </Step>
  <Step title="Create a Space">
    Click "New Space" and name it, like "Project Alpha Docs".
  </Step>
  <Step title="Add Content">
    Start with a page using the editor. Embed components for richer docs.
  </Step>
  <Step title="Invite Team">
    Share the space link or use API to automate invites via `https://api.example.com/teams`.
  </Step>
</Steps>
