## Setting Up Webhooks

Webhooks notify external services about events like document updates or space changes. Follow these steps to configure one.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to `https://dashboard.example.com/settings/webhooks` and click "New Webhook".
  </Step>
  <Step title="Configure Events" icon="settings">
    Select events such as `document.updated` or `space.published`.
  </Step>
  <Step title="Add Endpoint" icon="globe">
    Enter your webhook URL, e.g., `https://your-webhook-url.com/paperguide-pay`.
  </Step>
  <Step title="Test and Save" icon="check-circle">
    Send a test event and verify receipt.
  </Step>
</Steps>

Handle incoming webhooks securely. Verify signatures using the shared secret.

<CodeGroup tabs="Node.js,Python">
  ```javascript
  const crypto = require('crypto');

  app.post('/webhook', (req, res) => {
    const signature = req.headers['x-paperguide-pay-signature'];
    const payload = JSON.stringify(req.body);
    const expected = crypto
      .createHmac('sha256', process.env.WEBHOOK_SECRET)
      .update(payload)
      .digest('hex');

    if (crypto.timingSafeEqual(Buffer.from(signature), Buffer.from(expected))) {
      console.log('Verified:', req.body.event);
      res.status(200).send('OK');
    } else {
      res.status(401).send('Unauthorized');
    }
  });
  ```
  ```python
  import hmac
  import hashlib
  from flask import Flask, request

  app = Flask(__name__)
  WEBHOOK_SECRET = 'your-webhook-secret-here'

  @app.route('/webhook', methods=['POST'])
  def webhook():
      signature = request.headers.get('X-Paperguide-Pay-Signature')
      payload = request.get_data()
      expected = hmac.new(
          WEBHOOK_SECRET.encode(),
          payload,
          hashlib.sha256
      ).hexdigest()

      if hmac.compare_digest(signature, expected):
          print('Verified:', request.json['event'])
          return 'OK', 200
      return 'Unauthorized', 401
  ```
</CodeGroup>

## API Access for Custom Extensions

Access Paperguide-pay's REST API at `https://api.example.com/v1` to build custom integrations. Authenticate with a bearer token.

<ParamField header="Authorization" param-type="string" required="true">
  Bearer `{YOUR_API_TOKEN}` obtained from `https://dashboard.example.com/api-keys`.
</ParamField>

<ParamField query="spaceId" param-type="string" required="false">
  Filter by documentation space ID.
</ParamField>

<Tabs>
  <Tab title="List Documents" icon="list">
    Fetch all documents in a space.

    <Request tabs="cURL,JavaScript">
      ```bash
      curl -X GET "https://api.example.com/v1/spaces/{spaceId}/documents" \
        -H "Authorization: Bearer YOUR_API_TOKEN"
      ```
      ```javascript
      const response = await fetch('https://api.example.com/v1/spaces/{spaceId}/documents', {
        headers: {
          'Authorization': 'Bearer YOUR_API_TOKEN'
        }
      });
      const docs = await response.json();
      console.log(docs);
      ```
    </Request>
  </Tab>
  <Tab title="Create Document" icon="plus">
    Create a new document.

    <Request tabs="cURL,Python">
      ```bash
      curl -X POST "https://api.example.com/v1/documents" \
        -H "Authorization: Bearer YOUR_API_TOKEN" \
        -H "Content-Type: application/json" \
        -d '{
          "title": "New Doc",
          "content": "# Hello World",
          "spaceId": "{spaceId}"
        }'
      ```
      ```python
      import requests

      response = requests.post(
          'https://api.example.com/v1/documents',
          headers={
              'Authorization': 'Bearer YOUR_API_TOKEN',
              'Content-Type': 'application/json'
          },
          json={
              'title': 'New Doc',
              'content': '# Hello World',
              'spaceId': '{spaceId}'
          }
      )
      print(response.json())
      ```
    </Request>
  </Tab>
</Tabs>

<ResponseField name="data" field-type="array" required="true">
  Array of document objects.
</ResponseField>

<ResponseField name="documents[].id" field-type="string">
  Unique document identifier.
</ResponseField>

<Callout kind="tip">
  Rate limits apply: `<1000` requests per hour. Check headers for `X-RateLimit-Remaining`.
</Callout>
